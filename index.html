StokPro Kurulumu

Uygulamayı kullanmak için Google API bilgilerinizi girin.

1
Google Cloud Console'a gidin console.cloud.google.com

Yeni proje oluşturun veya mevcut projeyi seçin.

2
API'leri etkinleştirin

APIs & Services > Library'den: Google Sheets API ve Google Drive API

3
OAuth 2.0 Client ID oluşturun

Credentials > Create Credentials > OAuth client ID > Web Application
JS Origin: http://localhost:8080
